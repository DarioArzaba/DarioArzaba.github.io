7Zip

  • Herramientas
  • 30 Sep 2017
  • About 2 minutes

7Zip es una herramienta Open Source (GNU LGPL) diseñada para crear archivos comprimidos y desarrollada por [Igor Pavlov](https://en.wikipedia.org/wiki/Igor_Pavlov_(programmer%29). El programa genera archivos en formato 7z Format mediante el algoritmo de compresión LZMA.

Caracteristicas

  • Algoritmos de compresión LZMA, BZIP2, PPMD y DEFLATE.
  • Encriptación AES-256 para 7z.
  • Compresión aproximada de 1 MB/s en un CPU de 2 GHz.
  • Descompresión aproximada de 15 MB/s en un CPU de 2 GHz. El código descompresión es de aproximadamente 5 KB.
  • Capacidad para agregar comentarios a los archivos.
  • Formato 7z con soporte para archivos de 16,000,000 TB y compresión solida o por partes.
  • Uso del sistema de ordinamiento por nombre. Para reducir el tamaño del archivo se recomienda aumentar el del diccionario.
  • Errores suelen ocurrir por headers incorrectos, generalmente insertados por otros programas de compresion.

Archivos Instalados

Archivos en: C:\Program Files\7-Zip

  • Lang: Carpeta con los Lenguajes del programa (txt). Contiene el archivo (en.ttt) que administra las ventanas, advertencias y el menú emergente del programa.
  • 7zFM.exe: Ejecutable Principal con el GUI.
  • 7zG.exe: Ejecutable secundario utilizado por 7zFM para generar el GUI.
  • 7z.exe: Ejecutable para el modulo de comandos CMD.
  • descript.ion: Archivo que genera y guarda la función de Comentarios.
  • 7z.sfx: Modulo SFX (Self Extracting File) para (7zFM)
  • 7zCon.sfx: Modulo SFX (Self Extracting File) para (7z) es decir para el modo CMD.
  • 7z.dll: Engine Module. Libreria Principal y StandAlone PlugIn.
  • 7-zip.dll: Libreria para Windows Shell Extension
  • 7-zip32.dll: Libreria para Windows Shell x64
  • 7-zip.chm: Archivo de ayuda.

El archivo de ayuda contiene la lista actualizada de Formatos Compatibles, Métodos de Compresión. Performance del Programa, Herramienta Benchmark y los Switches de la versión CMD.

En 2008 Igor Pavlov publica al dominio publico el LZMA SDK nos permite desarrollar librerías basadas en el algoritmo de compresión LZMA. Este Kit contiene el código fuente del encoder y decoder LZMA para ASM, C, C++, C# y java.

Compresión

Al comprimir un archivo intentamos reducir su tamaño en la memoria. La compresión es Lossless cuando no se pierde información, sustituyendo las partes repetitivas por variables únicas que son guardadas y consultadas en un diccionario, por ejemplo el método LZW de Lempel, Ziv y Welch. La compresión es Lossy cuando se pierde parte de la información, en este caso no se sustituyen las partes repetitivas sino que se eliminan las partes innecesarias.
