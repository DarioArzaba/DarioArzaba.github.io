IRC

  • Programacion
  • 03 Mar 2017
  • About 2 minutes

IRC o Internet Relay Chat es un Protocolo de Internet para transferir mensajes de texto en un Chat Room o Channel (Canal), mismos que pueden ser accesados a travez de clientes IRC, de la misma forma que utilizamos clientes HTTP (Navegadores) para accesar a Internet.

Aunque es uno de los servicios mas antiguos del Internet su uso a decaido con la llegada de los mensajes instantaneos y las redes sociales, a pesar de ello sigue siendo usado para obtener ayuda en tiempo real para nichos de interes comun como Proyectos Open Source y Hobbies.

El mayor servidor de IRC es Freenode con alrededor de 100,000 usuarios activos. Algunos de los clientes mas populares son Textual o LimeChat para Mac y HexChat o mIRC para Windows. Existen algunos clientes adicionales para navegadores como KiwiIRC o Mibbit.

Servicios como BotBot archivan y decoran los canales de sus creadores para poder almacenar su actividad y que no se pierda al paso del tiempo ninguna idea importante.

Para unirse a un canal en Freenode se recomienda estar registrado con NickServ. Si queremos crear y registrar nuestro propio canal, Freenode tiene varios tutoriales. Mientras que NickServ registra los nombres de usuarios, ChanServ se encarga de personalizar los canales y realizar tareas automaticas. Usuarios con un arroba (@) en sus nombres significa que son OP u Operadores del canal y tienen permisos especiales.

Lista de Comandos para IRC:

Unirse a un canal (Crear si no existe):
/join #(Nombre del Canal)
Registrar un Nombre de Usuario:
/msg nickserv register (Contraseña) (Correo)
Iniciar Sesion:
/msg nickserv identify (Nickname) (Contraseña) 
Enviar una accion:
/me '(Accion)'
Crear un Chat Privado:
/msg (Nickname) (Mensaje)
Cambiar tu Nombre de Usuario
/nick (Nickname)
Enviar un Mensaje Privado:
/notice (Nickname) (Mensaje)
Abrir Ventana de Mensajes Privados:
/query (Nickname)
Salir de un canal:
/part
Salir de todos los canales:
/partall
Ver el ping (Latencia) con otro usuario:
/ping (Nickname)
Salir del Cliente:
/quit
Bloquear otro usuario:
/ignore (Nickname) 3
Desbloquear usuario:
/ignore -r (Nickname) 3
Ver Info de Otro Usuario:
/whois (Nickname)
Abrir un Chat Directo:
/chat (Nickname)
Obtener Ayuda
/help
Cambiar Titulo o Tema:
/topic #(Nombre del Canal) (Texto)
Quitar a un usuario del Canal:
/kick #(Nombre del Canal) (Nickname)
Limpiar la pantalla:
/clear
