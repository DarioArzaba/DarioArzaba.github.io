Kramdown

  • Programacion
  • 17 Feb 2017
  • About 4 minutes

Kramdown es una libreria en Ruby para procesar y convertir Markdown a HTML. Ademas de Kramdown puede interpretar Markdown, Github Flavored Markdown y HTML; exportando el contenido en forma de Kramdown, HTML, LaTeX y PDF. Para exportar en LaTex utiliza Mathjax.

Para poder utilizar Kramdown es necesario un interprete de Ruby, para instalar la gema correspondiente gem install kramdown

Referencia Rapida (Original):

  1. Titulos: Se usa un simbolo de gato para indicat el nivel del titulo.
  2. Parrafos: Todo string consecutivo separado por un salto de linea superior e inferior.
  3. Citas: Se inicia la cita con el simbolo de Mayor Que.
  4. Bloque de Codigo: Inicia y termina utilizando lineas curveas. Tambien se pueden utilizar parrafos indentados.
  5. Lineas: Para agregar lineas horizontales utilizamos tres asteriscos (***).
  6. Listas Ordenadas: Utilizan el numero, un punto y un espacio seguido del item.
  7. Listas Desordenadas: Utilizan asteriscos para cada item.
  8. Nested Lists: Se puede incluir cualquier elemento (incluso otras listas) dentro de un item de la lista siempre y cuando este indentado en su nivel.
  9. Definiciones: Utilizan un termino y en una nueva linea la definicion comenzando con un doble punto.
  10. Tablas: Utilizan el simbolo de pipa para separar cada columna.
  11. Atributos: Se introducen despues del elemento entre llaves e iniciando con dos puntos. Podemos utilizar una variable para definir una serie de atributos.
  12. Italics: Se encierra el texto entre asteriscos.
  13. Bold: Se encierra el texto entre dobles asteriscos.
  14. Links: Se pone entre corchetes el nombre y entre parentesis el URL.
  15. Imagenes: Se inicia con un signo de exclamacion seguido por el alt de la imagen entre corchetes y el Src entre parentesis.
  16. Codigo en Linea: Se encierra el codigo con backticks.
  17. Footnotes: Se encierra el numero entre corchetes, dicho identificador se referencia despues agregando un signo de dos puntos.
  18. Abreviaciones: Se colocan al final del texto con un asterisco y entre corchetes el texto de la abreviacion seguida de su definicion.
