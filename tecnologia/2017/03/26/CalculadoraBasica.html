Borra todos los ajustes y la memoria: Shift/9/3/= Para borrar solo la memoria Shift/9/2/= Para ajustar el contraste de la pantalla utilizar: Shift/Mode/Abajo/5. Después se utilizan las flechas laterales para ajustar el contraste. Aceptar el ajuste utilizando (AC).

La fx-82ES Plus Requiere de una batería triple AAA modelo Ro3 (UM-4). La cual dura aproximadamente 17,000 horas de vida consumiendo 0.0002W. La calculadora pesa 100 gramos. Para encender la calculadora se utiliza (ON). Para apagar la calculadora se utiliza (Shift + AC) La calculadora se apaga automáticamente después de 10 minutos de inactividad.

Indicadores:

  • S: Significa que se a presionado la tecla (Shift).
  • A: Significa que se ha presionado la tecla (Alpha).
  • M: Existe un valor en la memoria.
  • D: La unidad por defecto es Grados.
  • R: La unidad por defecto es Radianes.
  • G: La unidad por defecto es Gradianes.
  • STO: La calculadora está esperando un símbolo para utilizar como variable. Aparece después de activar la asignación de una variable (Shift RCL)
  • RCL: La calculadora está esperando el símbolo de una variable ya guardada para utilizar su valor. Aparece después de activar el generar variable (RCL).
  • STAT: La calculadora está en Modo de Estadística.
  • FIX: Se asignó un número máximo de decimales a generar en el resultado.
  • SCI: Se asignó un número máximo de dígitos totales a generar en el resultado.
  • Math: Se activó el uso de Natural Display (Como aparece en los apuntes).
  • (Flechas): Existe un historial al que se puede retroceder o adelantar.
  • Disp: La pantalla solo esta mostrando uno de los resultados de un cálculo múltiple.

Setup

  1. MthIO (Default): Formato para fracciones e irracionales. Aparecen como aparecerían en papel. Se puede seleccionar que el resultado sea también fraccionario o decimal (Prefiero Decimal)
  2. LineIO: Opción de formato para fracciones e irracionales. Aparecen en una sola línea de Input. Se activa automáticamente al entrar en el (Modo Stat).
  3. Deg (Default): Formato para la unidad angular. Establece el uso de grados.
  4. Rad: Formato para la unidad angular. Establece el uso de radianes.
  5. Gra: Formanto para la unidad angular. Establece el uso de Gradianes.
  6. Fix: Cantidad de decimales relevantes para el resultado. Establece un valor de 0 a 9 para la cantidad máxima de decimales a mostrar en el resultado. En 9 por default.
  7. Sci: Opción de cantidad de dígitos relevantes para el resultado. Establece un valor de 1 a 10 para la cantidad relevante de dígitos en un cálculo que resulta en notación científica.
  8. Norm: Opción de utilizar en el resultado notación científica o decimal cuando se está en el modo lineal. Cuando se está en el modo natural esta opción se selecciona en la misma opción 1.
  9. Ab/c: Establece el modo en el que se representan los resultados fraccionarios. Ya sea en forma mixta (Primer Opción) o Impropia (Segunda Opción, default).
  10. d/c: Establece si se activa o no la columna de frecuencias cuando se activa el modo STAT.
  11. STAT: Establece si se utiliza un Punto o una Coma para separar los decimales. Nótese que cuando Punto esta activado la separación de múltiples resultados en una computación múltiple se separan con comas, pero si se activa la Coma entonces estos resultados múltiples se separaran con puntos y comas.
  12. Disp: Ajusta el Contraste

Reglas Básicas

Los input tienen 99 bytes de memoria, casi toda operación y numero requiere de 1 byte aunque las funciones requieren de 3 hasta 13. El cursor cambia a un cuadrado cuando quedan menos de 10 bytes.

En el modo lineal uno puede cambiar el cursor de insertar por uno de remplazo utilizando (Shift/Del).

Para cambiar el resultado de fracción a decimal se puede usar el botón (SD). Luego, para cambiar de fracción impropia a mixta se puede utilizar (Shift/SD).

Para utilizar porcentajes se utiliza (Shift/(), por ejemplo para saber el 20% de 150 escribirmos: 150*20%=30. Por otro lado para saber que porcentaje representa un numero (660) de otro (880) utilizamos una division en vez de multiplicar 660/880%=75.

Para sumar o restar porcentajes (En impuestos o descuentos), por ejemplo para sumar el 15% de 2500 a 1000 escribimos: 1000+(2500*15%)=1375. Por otro lado para restar 25% a 3500 escribimos 3500-(3500*25%)=2625.

Para realizar cálculos múltiples se utiliza el separador de doble punto (:) en el input, esto dividirá las operaciones dando los resultados por separado. Por ejemplo cuando escribimos: 3+3:3*3=6,9.

Cuando un número está en notación científica podemos mover el punto decimal a la derecha tres lugares apretando (ENG) después de que se haya realizado el cálculo. Si queremos mover el punto decimal a la izquierda simplemente apretamos Shift/ENG.

La calculadora cuenta con las variables ABCDEFX y Y para que guardes cualquier valor numérico y que las puedas utilizar en tus cálculos. Por ejemplo para asignar el resultado de 3+5 a la variable A escibiriamos: 3+5(Shift/RCL)A=8. Ahora que se guardo el resultado 8 en la variable A la podemos utilizar en cálculos posteriores: A*10=80.

Para mostrar el contenido de la variable A simplemente utilizamos el botón (RCL) y la variable: (RCL) A = 8. Para limpiar los contenidos de la variable simplemente asignamos un cero a dicha variable. 0(Shift/RCL)A=0

Al calcular ángulos podemos representarlos en forma sexagesimal (Grados, Minutos y Segundos) se utiliza la tecla (.,,,) o Triple Coma (TC) la cual esta arriba de ENG. Por ejemplo si quisiéramos sumar 2 grados 20 minutos y 30 segundos con 39 minutos, 30 segundos realizaríamos lo siguiente: 2(TC)20(TC)30(TC)+0(TC)39(TC)+30(TC)= 3 Grados 0 Minutos 0 Segundos.

Para convertir un número sexagesimal a decimal o viceversa utilizamos el mismo símbolo del Triple Coma (TC) que esta arriba de ENG después de realizar la operación. Por ejemplo para pasar de 2 Grados 15 Minutos y 18 Segundo a decimal, escribimos lo siguiente: 2(TC)15(TC)18(TC) = 2G15M18S(TC) = 2.255. Para volver a transformar el resultado a sexagesimal simplemente volvemos a apretar (TC). 2.255(TC)=2G15M18S.

Memoria+: Uno puede insertar resultados directamente en la Memoria sin necesidad de utilizar variables utilizando la tecla (M). Para limpiar los contenidos de M también le asignamos un cero: 0(Shift/RCL)M+=0 Para añadir el resultado de 10*5 a la memoria: 10*5(M+)=50. Para restar 10+5 de la memoria (50) utilizamos 10+5(Shift/M+)=15. Para mostrar el contenido de la memoria simplemente utilizamos RCL: RCL M+ = 35.

Se puede factorizar un número (máximo 10 dígitos) en sus factores primos de hasta 3 dígitos. Por ejemplo para factorizar 1014 en sus factores primos: 1014(Shift/TC) = 2*3*13*13.

Si el número a factorizar tiene una parte que requiere un factor de mas de tres dígitos como por ejemplo 4104676 (2*2*1013*1013). Debido a que esta función no puede representarlo, en este caso (1013) va a dejar la parte que corresponde a este factor entre paréntesis: 4104676 = 4104676(Shift/TC) = 2*2*(1026169).

Funciones Extra

Para utilizar la función de número al azar (Ran#) primero se especifica el rango superior y después se utiliza (RAN#):

  • 1000(Shift + .) = Número integral al azar integral de menos de 1000.
  • 100(Shift + .) = Número al azar menor a 100 con un decimal.
  • 10(Shift + .) = Número al azar menor a 10 con dos decimales.
  • 1(Shift + .) = Número al azar entre 0 y 1 con tres decimales.

Para obtener números al azar en un rango más específico se utiliza (RANINT#): Alpha/RanInt#/1/Shift/)/6/)/=. Lo anterior obtiene un número al azar entre 1 y 6.

Para determinar permutaciones o combinaciones utilizamos (NPR) y (NCR) que son los Shift para multiplicación y división. Por ejemplo si quisiera saber el número de permutaciones o combinaciones al seleccionar 4 personas de un grupo de 10 escribiría: 10(Shift/*)4=5040 Permutaciones. 10(Shift//)4=210 Combinaciones.

Para pasar de coordenadas rectangulares a polares utilizamos (Pol) y (Rec). Por ejemplo para pasar de las coordenadas rectangulares (Raiz(2)+Raiz(2)) a su equivalente polar escribimos: Pol(Raiz2,Raiz2)=(r=2,Grados=45). Inversamente para pasar de polares a rectangulares utilizamos: Rec(Raiz2,45)=(x=1,y=1).

La calculadora elimina lo recursivo y lo redondea al resultado exacto, lo cual es lo correcto sin embargo si quisiéramos que nos mostrará el resultado aproximado en vez del exacto necesitamos utilizar (RND). Por ejemplo si quisiéramos que no redondeará la operación (10/3)*3=10 entonces escribimos: RND(10/3)*3=9.9999999.

Para insertar el símbolo de grados, radianes o gradianes respectivamente a nuestro input lo podemos realizar utilizando el botón DGR el cual es el Shift de ANS. Por ejemplo, (Shift/Ans/1)= Inserta símbolo de grados.

Crear una tabla: Seleccionar el modo de Tabla. Introduce una función utilizando la variable roja (X), recuerda que no se pueden insertar Pol o Rec. Cuando se termine de introducir el valor deseado presionar igual. Luego se preguntara el inicio y final de la tabla así como cuanto se debe avanzar en cada paso. Al finalizar y presionar igual se generara la tabla de valores X junto a la tabla de valores Y.

Modo Estadistica

Cuando entramos en este modo podremos realizar distintos tipos de fórmulas de regresión. Regresion de una variable, lineal, cuadrática, logarítmica, exponencial, AB, Potencia, Inversa.

Una vez que se seleccionó el tipo de operación que se requiere realizar se procede a introducir los datos. Se cuentan con 40 filas para introducir datos. Las filas se van agregando automáticamente, si se desea eliminar una fila simplemente se oprime Del. Si se quiere agregar una fila manualmente se oprime: (Shift/1)/3/1 Por otro lado para borrar todos los datos de la tabla se utiliza: (Shift/1)/3/2

Una vez que se terminó de crear la tabla se procede a oprimir (AC). Saldremos del editor de la tabla sin embargo el indicador de STAT nos dice que en la memoria existe una tabla a la que podemos aplicar una fórmula estadística: Para accesar las fórmulas estadísticas oprimimos: (Shift + 1)

Las fórmulas disponibles son:

  • Sum: Sumas Totales.
  • Variables: n (Número de datos), x (Media X), y (Media Y), Gx (Desviación estándar Poblacional) Sx (Desviación Estándar Muestra)
  • Reg: Regresión de Coeficientes A o B, Regresión del Coeficiente (r) Valores estimados de X y Y.

Por ejemplo para obtener la media y la desviación estándar primero entramos al modo estadístico (Mode/2), luego seleccionamos el tipo de regresión (1-Var), llenamos la tabla de valores de (X), y la de frecuencia.

Luego Podemos cambiar el tipo de regresión, los datos de la tabla o aplicar fórmulas con (Shift/1). En este caso queremos ocupar la formula de Media y Desviación que se encuentran en (Shift/1/4). Al escoger una fórmula simplemente ponemos (=) y se aplicará a la tabla guardada en memoria.

Errores

  • Math Error: El resultado excede el rango de calculación, el input excede el rango de input o la calculación contiene una operación ilegal (División entre Zero). La solución está en checar que no haya operaciones inválidas o reducir la cantidad de dígitos. (0/0)
  • Stack Error: Uso de paréntesis exesivo, volcó la memoria. La solución esta en reducir la cantidad de paréntesis, normalmente dividiendo la operación en dos partes.
  • Syntax Error: Faltan argumentos o argumentos no validos en las operaciones. Revisar el Input en busca del argumento no valido. (++)
  • Insufficient MEM: En el modo Table la calculación fue demasiado para que la memoria la pudiera guardar. La solución es reducir el rango o pasos de cálculo. (Range 0.1)
  • Argument Error: Un número no integral se intentó introducir como input de la función de generar numeros al azar(RANINT#). La solución es cambiar el argumento por uno integral. (RANINT 0.1)
